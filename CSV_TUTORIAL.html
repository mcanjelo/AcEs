Conversion from .xlsx to .csv format

All software have different limitations on the types of files they can work on. For this particular system, only Comma Delimited (.csv) files are accepted. In order to convert from the usual Microsoft Excel (.xlsx) format, just follow these steps:

  1. Open the file containing the class list you wish to upload.


  2. From the File tab, select Save As.


  3. From the new dialog box, notice the Save as type dropdown list. From that list, select CSV (Comma delimited) (*.csv). You may rename your file if you wish by typing in a new file name in the text field above the dropdown list.


  4. Click Save to finish the conversion process.
